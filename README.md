# wsdl-front

## Как развернуть на nginx (юзался nginx 1.23.4)

содержимое папки dist кидаем в /usr/share/nginx/html
юзаем стандартную конфигурацию. На всякий она лежит ./nginx.conf
для папки дист нужно сбилдить 

```
npm run build
```
